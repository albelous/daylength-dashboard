# Дашборд: длительность светового дня

Демо для практики по дашбордам.

## Демо
https://albelous.github.io/daylength-dashboard/

## Функции
- График 4 городов
- Выбор интервала дат
- Кнопки: 2 нед / нед / мес / 3мес
- Мобильная адаптация
- PDF экспорт (нужны библиотеки в libs/)

## Структура

├── index.html
├── assets/daylength_data.json
└── libs/ (html2canvas + jspdf)
